Objetivo da Fase de Projeto do Projeto -

Missão: A fase de projeto de teste é uma das fases mais criticas no desenvolvimento de sistemas computacionais, ela tem como objetico identificar os casos de testes mais adequados para cada funcionalidade do software, afim de identificar o maior numero de falhas possiveis.

Objetivo: Capacitar o aluno de maneira que o mesmo seja capaz de identificar quais casos de testes são mais adequados para caso de uso exercitando seu pensamento crítico acerca do desenvolvimento de sistemas computacionais.
